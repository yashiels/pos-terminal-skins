## Physical Buttons

```
         ┌──────────────┐
         │  (o) camera  │
         │              │
         │  ┌────────┐  │
         │  │ Screen │  │
         │  │720x1280│  │
  [SCAN] │  │        │  │ [PWR]
 orange  │  │        │  │ [SCAN]
         │  └────────┘  │  orange
         │              │
         │    SUNMI     │
         │    )))       │  <- NFC
         │   [USB-C]    │
         └──────────────┘
```

- **Power (right side, upper):** Sleep/wake, long-press for power menu
- **Right scan trigger (orange, right side, mid):** Barcode scanner trigger → maps to `camera`
- **Left scan trigger (orange, left side, mid):** Barcode scanner trigger → maps to `search`

## Other Hardware

- 5 MP rear camera (autofocus)
- 0.3 MP front camera (fixed focus)
- 2D barcode scanner (Sunmi SS1100)
- Magnetic stripe reader (top)
- IC chip card reader (bottom)
- NFC (front + rear dual antenna)
- USB-C port (bottom)
- Pogo pins for cradle charging
- No built-in printer (Lite variant)
